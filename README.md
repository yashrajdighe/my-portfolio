# Static Portfolio Website Template

A modern, responsive, single-page portfolio template built with HTML, Tailwind CSS, and vanilla JavaScript. It is designed to be easily customizable and deployable on any static hosting service.

## Features

- **Fully Responsive:** Adapts to all screen sizes.
- **Interactive UI:** Smooth scrolling, modals, and animations.
- **Key Sections:** Includes sections for About, Projects, Skills, Blog, Contact, and a Resume.
- **Easy to Customize:** Built with standard web technologies for straightforward editing.

## Technologies Used

- HTML5
- Tailwind CSS
- Vanilla JavaScript
- Lucide Icons
